This function takes from a 2D array each column and check if it is a fixed array a=[1,2,3,4,5,6,7,8,9]

#### `check_raws(table):`

This function takes from a 2D array each raw and check if it is a fixed array a=[1,2,3,4,5,6,7,8,9]
